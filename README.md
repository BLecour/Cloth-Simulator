# Cloth Simulator

A GPU-accelerated cloth simulator implemented using [NVIDIA Warp](https://github.com/NVIDIA/warp). Uses position based dynamics and produces a USD file with the render output. An example can be seen below. The cloth in this render contains 10,000 vertices.
